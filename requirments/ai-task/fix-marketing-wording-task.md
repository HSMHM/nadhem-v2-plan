# مهمة: تعديلات على محتوى وصياغة خطة التسويق

---

### 1. حذف "معدل التحويل" من المستهدفات

في `mktTargets` احذف:
```js
// احذف:
{ metric: 'معدل التحويل (Lead → عميل)', target: '20-25%', ... },
```

---

### 2. استبدال "لا يبحثون عنك"

في `challengePoints`:
```js
// القديم:
{ title: 'لا يبحثون عنك', desc: 'كثير من المنظمين يستخدمون أدوات يدوية (Excel، WhatsApp) ولا يعرفون وجود حلول رقمية', icon: 'eye-slash' },

// الجديد:
{ title: 'قلة المعرفة عن وجود حلول', desc: 'كثير من المنظمين يستخدمون أدوات يدوية (Excel، WhatsApp) ولا يعرفون أن هناك منصات متخصصة تسهّل عملهم', icon: 'eye-slash' },
```

---

### 3. تعديل "دورة البيع طويلة"

```js
// القديم:
{ title: 'دورة البيع طويلة', desc: 'من أول تواصل حتى توقيع العقد قد تمر 3-6 أشهر', icon: 'hourglass-half' },

// الجديد:
{ title: 'فترة التفاوض طويلة', desc: 'من أول تواصل حتى اتخاذ قرار الاشتراك قد تمر 3-6 أشهر خصوصاً في الجهات الكبيرة', icon: 'hourglass-half' },
```

---

### 4. حذف "السوق مبعثر"

في `challengePoints` احذف:
```js
// احذف:
{ title: 'السوق مبعثر', desc: 'المنظمون موزعون على مدن وقطاعات مختلفة — لا تجمعهم منصة واحدة', icon: 'map-pin' },
```

---

### 5. حذف نص المصدر

في `marketSize` احذف حقل `source`:
```js
// احذف:
source: 'المركز الوطني لتنمية القطاع غير الربحي + تقديرات السوق',
```

وفي المكوّن `ChallengeSection.jsx` — إذا كان يعرض نص المصدر تحت بطاقة حجم السوق، احذفه.

---

### 6. استبدال "نقاط الألم" بـ "نقاط الضعف"

ابحث في **كامل المشروع** عن `الألم` و `painPoints` وعدّل:

**في البيانات:**
```js
// في كل persona، غيّر اسم الحقل:
// القديم: painPoints
// الجديد: weakPoints

// أو أبقِ اسم الحقل كما هو لكن غيّر العنوان المعروض في المكوّن
```

**في المكوّن `PersonasSection.jsx`:**
```jsx
// القديم:
<h5>نقاط الألم</h5>

// الجديد:
<h5>نقاط الضعف</h5>
```

ابحث عن كل ظهور لكلمة "الألم" في ملفات `.jsx` و `.js` واستبدلها بـ "الضعف".

---

### 7. استبدال "للممولين" بـ "للمانحين"

ابحث في كامل المشروع عن "الممولين" أو "للممولين" واستبدل بـ "المانحين" أو "للمانحين".

مثال في شريحة المنظمة غير الربحية:
```js
// القديم:
'عدم وجود تقارير للممولين'
// الجديد:
'عدم وجود تقارير للمانحين'
```

---

### 8. توسيع شريحة "الشركة التي تقيم فعاليات داخلية"

عدّل هذه الشريحة لتشمل الجهات الحكومية أيضاً:

```js
// القديم:
{
  name: 'الشركة التي تقيم فعاليات داخلية',
  icon: 'building',
  desc: 'شركات كبرى ومتوسطة تقيم مؤتمرات أو معارض سنوية...',
  ...
},

// الجديد:
{
  name: 'جهات تقيم فعاليات داخلية',
  icon: 'building',
  desc: 'شركات وجهات حكومية تقيم مؤتمرات أو ملتقيات سنوية (مثل: ملتقى الموظفين، مؤتمر الشركاء، يوم المنتج). لا تنظم فعاليات كمهنة لكنها تحتاج أداة احترافية لإدارة التنظيم.',
  size: 'آلاف الجهات (حكومية وخاصة)',
  weakPoints: ['لا خبرة في تنظيم الفعاليات', 'الحاجة لحل سريع بدون تعقيد', 'صعوبة التنسيق بين الأقسام الداخلية'],
  priority: 'medium',
},
```

### 11. استبدال كلمة "ستضيع"

في فجوة CRM:
```js
// القديم:
desc: 'يوجد نظام CRM لكنه غير مفعّل — بدون تفعيله ستضيع بيانات العملاء المحتملين ومراحل المتابعة'

// الجديد:
desc: 'يوجد نظام CRM لكنه غير مفعّل — تفعيله يساعد في تنظيم بيانات العملاء المحتملين ومتابعة مراحل التواصل بشكل أفضل'
```

ابحث أيضاً عن أي ظهور آخر لكلمة "ستضيع" أو "تضيع" في سياق سلبي واستبدلها بصياغة إيجابية.

### 12. صفحة هبوط واحدة بدلاً من 3

في `mktTasks.marketingTeam`:
```js
// القديم:
{ id: 'MKT.12', task: 'إنشاء 3 صفحات هبوط مخصصة (غير ربحي / حكومي / خاص) وربطها بالإعلانات', ... },

// الجديد:
{ id: 'MKT.12', task: 'تحسين صفحة الهبوط الحالية لتخاطب جميع القطاعات مع ربطها بالحملات الإعلانية', frequency: 'مرة واحدة ثم تحديث ربع سنوي', duration: '2 يوم', priority: 'high' },
```

في `mktTasks.productManager`:
```js
// القديم:
{ id: 'PM.12', task: 'إعداد صفحة هبوط مخصصة لكل قطاع (غير ربحي / حكومي / شركات)...', ... },

// الجديد:
{ id: 'PM.12', task: 'مراجعة محتوى صفحة الهبوط الحالية وتحسينها لتناسب جميع القطاعات', frequency: 'مرة واحدة ثم تحديث ربع سنوي', duration: '1 يوم', priority: 'high' },
```

في `mktTimeline` — استبدل "إطلاق 3 صفحات هبوط مخصصة" بـ:
```
'تحسين صفحة الهبوط الحالية لتناسب جميع القطاعات'
```

### 14. جدول البحث الشامل عن التناقضات

بعد تطبيق كل التعديلات أعلاه، ابحث في **كامل المشروع** (`src/data/` + `src/components/`):

| ابحث عن | الإجراء |
|---------|--------|
| `الألم` | استبدل بـ `الضعف` |
| `للممولين` أو `الممولين` | استبدل بـ `للمانحين` أو `المانحين` |
| `ستضيع` أو `تضيع` (سياق سلبي) | أعد الصياغة بشكل إيجابي |
| `لا يبحثون عنك` | استبدل بـ `قلة المعرفة عن وجود حلول` |
| `السوق مبعثر` | احذف |
| `دورة البيع` | استبدل بـ `فترة التفاوض` |
| `توقيع العقد` (في سياق التسويق) | استبدل بـ `اتخاذ قرار الاشتراك` |
| `3 صفحات هبوط` | استبدل بـ صفحة واحدة محسّنة |
| `بريد إلكتروني بارد` أو `Cold Email` | استبدل بـ `التواصل عبر البريد الإلكتروني` |
| `1,910 جمعية` | استبدل بـ `100+ جمعية تقيم فعاليات` |
| `شراكة إحالة مع أقسام المبيعات` | استبدل بالصياغة الجديدة |
| `معدل التحويل` (في المستهدفات) | احذف |
| `المصدر:` أو `source:` (في marketSize) | احذف |
| `برنامج إحالات` | تأكد أنه محذوف بالكامل (من المهمة السابقة) |
