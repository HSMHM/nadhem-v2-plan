# مهمة: تعديل أسعار الباقات وتوزيع الوحدات

## التعديلات

### 1. تعديل الأسعار في `src/data/packages.js`

| الباقة | السعر القديم | السعر الجديد |
|--------|-------------|-------------|
| أساسية (Starter) | 12,000 ر.س | **4,000 ر.س** |
| متقدمة (Professional) | 25,000 ر.س | **8,000 ر.س** |
| شاملة (Enterprise) | 45,000 ر.س | **15,000 ر.س** |
| مخصصة (Custom) | حسب الطلب | **عرض سعر** |

### 2. تعديل وحدات الباقة المتقدمة (Professional)

احذف هاتين الوحدتين من الباقة المتقدمة:
- **إدارة الجهات العارضة** (Module index 2)
- **إرسال وإدارة الدعوات** (Module index 13)

### 3. تعديل وحدة "زيادة المساحة"

في `src/data/modules.js`:
```js
// القديم:
{ name: 'زيادة المساحة (تخزين)', icon: 'hard-drive', note: '' },
// الجديد:
{ name: 'زيادة المساحة (50 جيجا)', icon: 'hard-drive', note: '' },
```

نفس التعديل في أي مكان آخر تظهر فيه "زيادة المساحة (تخزين)" — ابحث في كل المشروع.

### 5. تحديث المصفوفة في `PackagesSection.jsx`

تأكد أن جدول المصفوفة (الوحدات × الباقات) يعكس التغييرات — الباقة الاحترافية لم تعد تحتوي إدارة الجهات العارضة ولا إرسال الدعوات.

### 6. تحديث نفس البيانات في قسم خارطة الطريق

إذا كانت الباقات مذكورة أيضاً في `src/data/roadmap.js` أو في `RoadmapSection.jsx`، حدّثها بنفس الأسعار والوحدات.

### 7. البحث الشامل

ابحث في كامل المشروع عن القيم القديمة:
```
12,000 → لا يجب أن تظهر كسعر باقة
25,000 → لا يجب أن تظهر كسعر باقة
45,000 → لا يجب أن تظهر كسعر باقة
"حسب الطلب" → استبدل بـ "عرض سعر" (في سياق الباقات)
"زيادة المساحة (تخزين)" → "زيادة المساحة (50 جيجا)"
```
